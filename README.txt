Archivos para despliegue:
- index.html
- sello.png

Despliegue en Vercel:
1. Cree un proyecto nuevo en Vercel.
2. Suba esta carpeta completa o arrástrela al panel de Vercel.
3. Vercel detectará un sitio estático y publicará index.html automáticamente.

Despliegue en Netlify:
1. Entre a Netlify y elija "Add new site" o arrastre esta carpeta.
2. Suba la carpeta completa con index.html y sello.png.
3. Netlify publicará el sitio como página estática.

Notas:
- No requiere backend ni base de datos.
- La impresión funciona desde el navegador usando "Imprimir" o "Guardar como PDF".
- Mantenga index.html y sello.png en la misma carpeta.
